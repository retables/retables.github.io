Examples context

Tools used for the generated data​

The documentation's examples will use a mocked set of people, generated with the @faker-js/faker.
You can see the complete array at the bottom of this page.

note

Some examples, for semplicity only use a small subset of the people array

Demo of the results​

Various sections will present live demos of how the library works. Keep in mind that every table you see is created using reTables.

Brief note about css​

The example components in the documentation uses Tailwind.
However, nothing prevents you from using the classic css or any other framework.

The person type​

src/types/index.ts
export type Person = {
    id: number;
    about: {
        firstName: string;
        lastName: string;
        job: string;
        age: number;
    };
    contacts: {
        address: string;
        phone: string;
    };
    company: string;
};
